## 当前入口约定

- 用户和运维入口：`launch_scripts/robot.sh`
- 工程编排入口：`src/robot_bringup/launch/system.launch.py`
- 默认导航流程：`./robot.sh navigation ...`
- 两阶段导航：仅作为回退和诊断模式
