# 上位机与下位机关联文档整理需求冻结

## 目标

整理当前上位机 ROS2 与下位机 STM32/ESP01S 的关联文本文档，形成一个可稳定访问、可追溯、能指导后续排查的主入口。

## 交付物

- 新增 `docs/host-mcu-integration-status.md`
- 更新 `docs/README.md`
- 新增本轮 `$vibe` 计划与运行回执

## 约束

- 不运行构建、ROS、topic/node/tf 或硬件命令。
- 不重命名旧中文文档，避免 SMB/编码问题扩大。
- 不声称远程验证通过。
- 文档必须明确区分静态已修正项和仍需远程验证项。

## 验收标准

- 文档能串联串口、速度尺度、航向/IMU、里程计、控制仲裁、导航融合。
- 文档能关联到上位机和下位机关键文件。
- 文档能索引已存在的 `requirements`、`plans`、`vibe-sessions`。
- 文档包含远程机器人验证边界与命令模板。
