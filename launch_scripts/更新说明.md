### 🚀 使用方法

#### 启动摄像头（单独）
```bash
cd /home/robot/ros2_ws/launch_scripts
./start_camera.sh
```

#### 启动完整系统（包含摄像头）
```bash
cd /home/robot/ros2_ws/launch_scripts
./start_full_system.sh
```

#### 桌面方式
- 双击"📷 启动摄像头"图标
- 或双击"🤖 完整系统"图标
